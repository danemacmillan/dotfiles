# Dotfiles

Dotfiles that work seamlessly between OSX and NIX environments.

This is the first iteration. It is messy.

## Installation

`git clone https://github.com/danemacmillan/dotfiles.git .dotfiles && cd .dotfiles && source bootstrap.sh`
